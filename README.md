## Required

installed [babashka](https://github.com/babashka/babashka)

```bash
$ bash < <(curl -s https://raw.githubusercontent.com/babashka/babashka/master/install)
```

## Usage

```bash
// prepare data
$ history -i | awk '{print $2,$3,$4,$5,$1}'  > input.txt

$ bb report.clj
```
